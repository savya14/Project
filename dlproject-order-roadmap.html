Month 1–2 · Project 1
⚽ Expected Goals (xG) Model
Beginner-friendly

Use real professional shot data from StatsBomb — the same data provider used by FC Barcelona — to train a model that predicts the probability of any shot resulting in a goal. This is the xG metric you see on Sky Sports, built from scratch by you. Add SHAP explainability charts so you can answer "why did this shot have a 0.3 xG?" in an interview. Deploy as a Streamlit app with a player selector and shot map.

"The interesting part was comparing my model's xG against Understat's official numbers for La Liga strikers."

Month 2–3 · Project 2
💰 Transfer Value Estimator
Intermediate

Scrape player market values from Transfermarkt and combine with stats from FBref to predict what any player should be worth. Then layer NLP on top — scrape transfer news headlines and run FinBERT (a finance-tuned BERT model) to detect whether media buzz is positive or negative, and adjust the prediction by up to ±15%. Three distinct skills in one pipeline: scraping, regression, and NLP.

"The name mismatch between Transfermarkt and FBref was a real data engineering problem — I solved it with Python's difflib fuzzy matching."

Month 3–4 · Project 3
🎥 Football Tactics Analyzer
Advanced

Take a real match video clip and run YOLOv8 — the same object detection model used in self-driving cars — to detect every player on the pitch. Classify players into two teams using K-Means clustering on jersey colors. Apply a homography transformation to map all player positions onto a 2D bird's-eye pitch view. Generate movement heatmaps and auto-detect the formation (e.g. "4-3-3"). Record a 60-second demo and post it on LinkedIn — this is the one people stop to watch.

"In the interview, I showed the live demo on my laptop. The interviewer asked me to explain homography. That became a 10-minute conversation."
